Vad är en regel i CSS och vad består den av?

CSS regel bestämmer elementets utseende på sidan. Regeln består av en selektor och en eller flera deklarationer. Selektorn bestämmer elementet på vilket regeln ska tillämpas, medans deklarationen bestämmer elementets egenskaper.

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Om man vill adressera ett enda element i en CSS-regel, ska man använda sig av ett id. Detta är ett globalt attribut som kan sättas för alla HTML-element. Ett id ska vara unikt och får bara förekomma en enda gång i varje HTML-dokument.

Exempel #idnamn { color: blue; }

Till skillnad från id kan klass tillämpas på flera element.

Exempel .klassnamn { color: blue; }
